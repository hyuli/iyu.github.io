Lock中的各种lock方法

2016-05-29
Java Concurrency
Java Concurrency


如图，在Lock类中有这么多lock方法，他们分别有什么区别呢。

  1. lock(), 比较流氓的方式，一直尝试去获取锁，不响应interrupt().
  2. tryLock(), 马上返回，拿到锁就返回true，否则false。
  3. tryLock(long time, TimeUnit unit)，比较聪明的方式，会等待一段时间，超时就返回false。
  4. lockInterruptibly(), 如果线程阻塞时，收到interrupt会被唤醒并要求去处理interrupt exception.
